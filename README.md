## Tarea 1 - Ejemplos de software

**Buen software → Netcat (nc)**
La flexibilidad que te da para conectarte a servidores desde la propia terminal de Linux y la seguridad con la que lo hace, entre otras características.

**Mal software → Microsoft PowerShell**
Lo tuve que usar como alternativa a la consola de Linux para una competición de ciberseguridad y me dio más problemas que soluciones, debido a la poca flexibilidad que tienes sin ser admin y lo poco intuitivo que resulta en comparación a la consola de Linux.

⏱️ **Tiempo empleado: 10 min** — Decidir entre las herramientas usadas y qué características señalar.

## Tarea 3 - Hoja de firmas desde CSV

Valiéndose de IA, a partir de un `.csv` obtener una hoja de firmas `.pdf` (Python):

- Leer el `.csv`
- Escribir el encabezado
- Escribir los alumnos con formato: Nombre, Apellidos, Lunes, Jueves
  - En grupos de 8
  - Ordenados por nombre alfabéticamente
  - Con tests unitarios
- Poner el código bajo control de versiones
- Publicar el código

⏱️ **Tiempo empleado: 2 min** — Lo que tardó la IA en generar el código, verificarlo y publicarlo.
